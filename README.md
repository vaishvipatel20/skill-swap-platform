# 💼 Career Intelligence and Recruitment Management System

*An Odoo-based enterprise recruitment platform developed during the Odoo Hackathon to automate hiring workflows, streamline candidate management, and improve recruitment efficiency.*

---

# 📖 Project Overview

The **Career Intelligence and Recruitment Management System** is an enterprise web application developed using the **Odoo ERP Framework** to modernize and automate the recruitment process. The platform provides a centralized environment where organizations can efficiently manage job postings, candidate applications, recruitment workflows, and hiring activities, while enabling candidates to explore opportunities and apply for positions through a simple and intuitive interface.

Developed using **Python**, **Odoo**, **PostgreSQL**, **XML**, **HTML**, **CSS**, and **JavaScript**, the application follows a modular architecture that supports secure data management, role-based access, and business process automation. The system minimizes manual administrative tasks, improves communication between recruiters and applicants, and enables organizations to manage the complete recruitment lifecycle from job creation to candidate selection through a single integrated platform.

This project demonstrates the practical implementation of **Enterprise Resource Planning (ERP)** concepts, full-stack web development, database management, and business workflow automation. It highlights the ability to design scalable enterprise applications that solve real-world organizational challenges while improving operational efficiency and user experience.

---

# 🎯 Project Objective

The primary objective of this project is to develop a centralized recruitment management platform that simplifies hiring processes for both organizations and job seekers. The system enables recruiters to efficiently create job postings, manage candidate information, monitor recruitment progress, and organize hiring activities, while providing applicants with a seamless experience for exploring opportunities and submitting applications.

The project also aims to demonstrate the effective use of the Odoo ERP framework in automating business processes, improving data organization, reducing manual effort, and enhancing decision-making through an integrated enterprise application.

```text

      ↓
Web Portal
      ↓

      ↓
Odoo ERP Backend
      ↓
PostgreSQL Database
      ↓
Recruiter Dashboard
      ↓
Candidate Management & Hiring Process
```

The application provides a centralized platform where candidates can apply for jobs, while recruiters manage applications, monitor recruitment progress, and organize hiring activities through the Odoo ERP system.

---

# ⚙️ Working Principle

The system allows administrators to create and manage job postings through the Odoo dashboard. Candidates can browse available positions, submit applications, and maintain their profiles. Submitted information is securely stored in the PostgreSQL database, where recruiters can review applications, update candidate status, and manage the complete recruitment workflow using dedicated ERP modules.
